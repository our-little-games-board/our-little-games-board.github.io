## 🎮 游戏介绍

这是一款五子棋小游戏，采用 HTML 和 JavaScript 构建，界面简洁、操作流畅，支持双人本地对战。

- 支持黑白双方轮流落子  
- 点击棋盘落子，规则标准  
- 适配桌面和手机浏览器
